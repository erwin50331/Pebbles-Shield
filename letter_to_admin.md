# 致伺服器管理員

你好，

我是頻道管理員（FBI）。我正在為「黑天鵝」社群開發一套自動化防護系統，名稱為「鵝卵石之盾（Pebbles Shield）」。

## 這個 Bot 是做什麼的？

網路次文化惡意隱語（例如淫夢梗、歧視性諧音詞）演變非常快，靠人工一個一個審查已經跟不上速度。這個 Bot 的目的是**自動偵測並記錄**伺服器內出現的惡意詞彙，讓我能更快速、有效地維護社群環境。

Bot 的主要功能：

- **即時監控**：自動掃描頻道訊息，比對已知惡意詞彙黑名單
- **違規記錄**：將疑似違規訊息記錄到管理員專屬的 Log 頻道，不會自動刪除或處罰（由人工判斷）
- **趨勢追蹤**：定期從 PTT 等平台抓取新興黑話，補充黑名單
- **管理介面**：提供儀表板讓管理員審核、一鍵處理違規

## Bot 需要哪些權限？

| 權限 | 用途 |
|------|------|
| Read Messages / View Channels | 讀取頻道訊息以進行比對 |
| Read Message History | 補抓加入前的訊息紀錄 |
| Manage Messages | 必要時刪除違規訊息 |
| Kick Members | 執行踢出操作 |
| Ban Members | 執行封禁操作 |

> 以上所有操作均由管理員在儀表板手動確認後才執行，Bot 不會自動懲處任何人。
